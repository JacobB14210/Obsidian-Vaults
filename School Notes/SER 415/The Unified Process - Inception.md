# Iterative Development and RUP
- Inception: Understand what to build
	- Vision, high level requirements
	- Not detailed
- Construction: Build the product
	- Working product, system test complete
- Transition: Validate solution
	- Stakeholder acceptance
# Inception: Know what to Build
1. Prepare vision document and initial business case
	- Include risk assessment and resource estimate
2. Develop "high-level" project requirements
	- Initial scenarios and domain models
3. Manage project scop
	- Reduce risk by identifying all key requirements
	- Manage change
